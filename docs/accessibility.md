# Accessibility notes

- The chat composer supports keyboard submit with `Enter` and newline with
  `Shift+Enter`.
- TTS controls expose labels and disabled states.
- Source citations are links with visible numeric labels.
- Motion is limited to lightweight loaders and should respect future
  `prefers-reduced-motion` refinements.

Recommended next checks:

- axe or Playwright accessibility scan.
- Keyboard-only walkthrough.
- Contrast review after any design-system token upgrade.
