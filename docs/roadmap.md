# Roadmap

## Current MVP

- Public web demo on Vercel.
- Python LangGraph agent.
- Mistral Document Library RAG via Mistral Agents API.
- Mistral library/admin ingestion script for PDF/file sources.
- Mistral Voxtral TTS.
- AXA-like UX and Canopée design-system integration.

## Next platform increments

1. Dedicated guardrails service with policy tests.
2. Automated RAG evaluations and golden-answer datasets.
3. OAuth2/OIDC and tenant-aware authorization.
4. OpenTelemetry traces and Dynatrace export.
5. MCP tools for controlled enterprise system actions.
6. A2A-compatible agent handoff for specialized prevention, claims and fleet
   assistants.
7. Azure/OpenShift deployment reference with IaC.
8. MLflow/Langfuse evaluation and prompt release management.
