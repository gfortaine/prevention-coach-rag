# Design system

The UI uses AXA France Canopée in `prospect` mode:

- `@axa-fr/canopee-react`
- `@axa-fr/canopee-css`
- Prospect tokens and reboot CSS
- Components currently used: `Button`, `Link`, `Tag`

The chat bubbles, preprompt buttons, source citations, TTS controls and guide
viewer remain custom. This is deliberate: those surfaces were tuned to match
the behavior of the public AXA Prevention assistant iframe, while keeping the
repository independent and transparent.
